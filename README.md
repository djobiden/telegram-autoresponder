# autoresponder for Telegram
## Simple autoresponder for Telegram 

### Installation 
```
git clone https://github.com/djobiden/telegram-autoresponder.git
```

### setup-file
> cfg.json
```
{
  "api_id": "your_api_id",
  "api_hash": "your_api_hash",
  "name": "bot_name",
  "message": "messages to reply"
}
```
